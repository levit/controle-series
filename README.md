## Sobre o projeto

Esse é um projeto de curso Alura para atualização Laravel.

## Sobre o curso

Laravel é um framework PHP livre e open-source criado por Taylor B. Otwell para o desenvolvimento de sistemas web que utilizam o padrão MVC.

Nesta formação você vai criar aplicações web utilizando o framework, para agilizar e facilitar o seu trabalho no dia a dia.

Link para o curso *[Formação Laravel](https://cursos.alura.com.br/formacao-laravel)*
